# 🦸‍♂️ Sortable Superheroes Table

A vanilla JavaScript project to display, search, sort, and paginate a dataset of superheroes from a public API.

---

## 📌 Project Overview

This project builds an interactive web page that organizes superhero data from a JSON API.  
No frameworks are allowed — everything is built using **HTML, CSS, and JavaScript only**.

---

## 🚀 Features

### 📊 Table Display
- Displays superheroes in a table format
- Shows:
  - Image
  - Name
  - Full Name
  - Powerstats
  - Race
  - Gender
  - Height
  - Weight
  - Place of Birth
  - Alignment

---

### 🔎 Search
- Live search by name
- Filters results on every keystroke
- Case-insensitive matching

---

### 🔃 Sorting
- Click on table headers to sort
- Supports:
  - Ascending / Descending toggle
  - Alphabetical and numerical sorting
- Missing values are always placed last

---

### 📄 Pagination
- Page size selector:
  - 10 / 20 / 50 / 100 / All
- Default page size: **20**

---

## ⚡ Performance
- Optimized filtering and sorting
- Designed to handle large datasets smoothly

---

## 📡 Data Source

Superhero dataset is fetched using `fetch`:
